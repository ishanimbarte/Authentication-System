# MERN Authentication App

## Features
- User Signup/Login
- Image Upload with Cloudinary
- JWT Authentication
- MongoDB Database

## Installation

1. Clone the repository

git clone (https://github.com/ishanimbarte/Authentication-System.git)

2. Install dependencies

cd backend
npm install

cd ../frontend
npm install

3. Create .env file

Add environment variables.

4. Start backend

node server.js

5. Start frontend

npm run dev
